Key Idea 1 : surface divide domain

divide domain using U and V parameter.

you should have use divide domian2 and isotrim.



When you project \( u \) and \( v \) onto the surface as if it were laid flat, it’s helpful to think of them as corresponding to the \( x \)- and \( y \)-axes. The components most commonly used for dividing a surface are **Divide Domain2** and **Isotrim**; these two components often work in tandem. The domain type outputs the parameters \( u \) and \( v \) \(\{interval\: 0 \text{ to } max\}\), indicating where the surface will be divided. This interval type is unique to the specific surface. By inputting this interval type into the **Isotrim** component, you generate a surface corresponding to that specific segment. The resulting surface is an untrimmed surface, not a split one, which makes it more versatile for further use.

private void RunScript(List<Surface> x, int y, ref object A, ref object B)
{
    List<Curve> contour = new List<Curve>();

    // Iterate over each surface to extract specific edge curves
    for (int i = 0; i < x.Count; i++)
    {
        // Convert surface to Brep and duplicate edge curves
        var crv = x[i].ToBrep().DuplicateEdgeCurves();
        contour.Add(crv[2]);
        if (i == x.Count - 1) contour.Add(crv[1]);
    }

    List<Curve> vertical = new List<Curve>();
    Curve st = contour[0];
    // Divide the start curve into points based on the count 'y'
    var pts = DividePts(st, y, true);

    // Loop through the contour curves to generate vertical lines
    for (int i = 0; i < contour.Count - 1; i++)
    {
        var now = contour[i];
        var nxt = contour[i + 1];
        List<Point3d> nxtPts = new List<Point3d>();

        // Generate corresponding points on the next curve
        for (int j = 0; j < pts.Count; j++)
        {
            double t;
            nxt.ClosestPoint(pts[j], out t);
            var nxtPt = nxt.PointAt(t);
            nxtPts.Add(nxtPt);

            // Add vertical lines alternately based on curve indices
            if (i % 2 == 0)
            {
                if (j % 2 == 0) vertical.Add(new Line(pts[j], nxtPt).ToNurbsCurve());
            }
            else
            {
                if (j % 2 == 1) vertical.Add(new Line(pts[j], nxtPt).ToNurbsCurve());
            }
        }

        pts = nxtPts;
    }

    B = vertical;
    A = contour;
}

// Method to divide a curve into a set number of points
public static List<Point3d> DividePts(Curve crv, int cnt, bool end)
{
    List<Point3d> ret = new List<Point3d>();
    var crvT = crv.DivideByCount(cnt, end);
    for (int i = 0; i < crvT.Length; i++)
    {
        var pt = crv.PointAt(crvT[i]);
        ret.Add(pt);
    }

    return ret;
}

2. Divide the first curve into points based on the input count 'y'.


3. Loop through the extracted curves and generate corresponding points on the next curve.


4. Add vertical lines alternately based on curve and point indices.


5. Store the generated contour and vertical lines in output variables.


6. Use helper functions to divide curves into points for vertical line generation.
