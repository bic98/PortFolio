Key Idea 1 : The map and filter function in Python.

Understanding the Python map() Function:

The map() function in Python is used to apply a specified function to each item of an iterable (like a list or tuple) and return a map object (which is an iterator). The map() function takes two arguments: the function to apply and the iterable. It allows you to process each element of the iterable according to the function provided, without having to write an explicit loop.

For example:


def square(x):
    return x * x

numbers = [1, 2, 3, 4, 5]
squared_numbers = map(square, numbers)
print(list(squared_numbers))
# Output: [1, 4, 9, 16, 25]

In this example, the map() function applies the square function to each element in the numbers list, resulting in a new list of squared values.

Understanding the Python filter() Function:

The filter() function in Python is used to construct an iterator from elements of an iterable (such as a list, tuple, or string) for which a specified function returns true. It takes two arguments: the first is the function that tests whether an item should be included in the result, and the second is the iterable whose elements are to be filtered.

For example:


def is_even(num):
    return num % 2 == 0

numbers = [1, 2, 3, 4, 5, 6]
even_numbers = filter(is_even, numbers)
print(list(even_numbers))
# Output: [2, 4, 6]

In this example, the filter() function applies the is_even function to each element in the numbers list. Only the elements for which the is_even function returns True are included in the resulting filtered list.
